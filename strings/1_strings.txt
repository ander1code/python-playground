Python - String Formatting

	String formatting in Python is the process of building a string representation 
	dynamically by inserting the value of numeric expressions in an already existing string.
	
	Using % operator
	
		Exemplo 01:
		
			s1 = "Hello, "
			s2 = "World!"
			print("%s%s" % (s1, s2))
	
	Using format() method
	
		Exemplo 01:

			s1 = "Hello, "
			s2 = "World!"
			print("{}{}".format(s1,s2))
			
	Using f-string
	
		Exemplo 01:
	
			s1 = "Hello, "
			s2 = "World!"
			print(f"{s1}{s2}")
				
	Using String Template class
	
		The String Template class belongs to the string module and provides a way to format 
		strings by using placeholders.
	
		Exemplo 01:
		
			from string import Template
			message = "Hello, $name!"
			template = Template(message)
			message = template.substitute(name="Anderson")
			print(message)
			

Python - Escape Characters

	An escape character is a character followed by a backslash (\).
	
	Unless an 'r' or 'R' prefix is present, escape sequences in string and bytes literals 
	are interpreted according to rules similar to those used by Standard C.
	
	Exemplo 01:
	
		print(r"\nHello!")
		print(R"\nHello!")

		\f				Alimenta√ß√£o de formul√°rio (ASCII FF)
